# Changelog

## 0.1.0 (2026-03-22)

- Initial release
- Authenticated client with session persistence
- Bookings API (list, create, edit, delete, availability, cost)
- Customers API (list, details, notes, blocking)
- Rooms API (list, lookup, descriptions)
- Scheduler API (calendar events, quick view)
- Admin API (activity types, pricing, audits, reports)
- Typed dataclass models
- CLI tool for common operations
